Section 04
Application Playbook
This week: Sell your timezone, don't apologize for it
Most LatAm tech workers (Colombia, Mexico, Argentina) overlap 4–8 hours with US Eastern time. That's a feature, not a bug. US hiring managers worry about async delays — remove that worry in the first paragraph.

Write this:
"I'm based in Bogotá (COT, UTC-5) — I have full overlap with US ET mornings and can be available for PT afternoons when needed." Not this:
"I'm in a different time zone but I'm flexible and can adjust my schedule as needed." One projects confidence. The other plants doubt. Same person, completely different first impression.

💡 En español: No te disculpes por tu zona horaria — véndela. Colombia y México tienen overlap total con el horario ET de la mañana. Mencionalo en el primer párrafo de tu cover letter.
Section 05
Skill Move
This week: Build the portfolio project that opens AI engineering doors
The most-requested AI role right now is not ML engineer — it's someone who can connect AI to business workflows. The fastest way to prove that is a portfolio project.

Build this: Customer support AI triage bot
• Input: a customer support ticket (text)
• Routing logic: classify the issue type using an LLM
• Output: suggested response, priority, and team to route to
• Tools: Python + Anthropic/OpenAI API + a simple UI
• Bonus: show a before/after on resolution time

Put it on GitHub. Write a short README. Add screenshots. This is the project that gets you past "AI curious" and into "AI capable" — which is where the $90K+ roles live.

Esta semana: construye un bot de triage para soporte al cliente usando un LLM. Ponlo en GitHub con un README claro. Ese proyecto te abre puertas a roles de +$90K que requieren experiencia en AI aplicado.
